# PUMA Dataset

PUMA: 3,195 Yahoo! Answers health questions, 10,067 answers.

**Not included.** Request access from dataset organizers:
- PUMA-PLASMA: m.schuiveling@umcutrecht.nl
- Or check the PUMA-PLASMA / Naik et al. 2024 repo for instructions

Place the data file as `puma.jsonl` in this directory, or set `PUMA_DATA_PATH` to its path.

Expected format: JSONL with `id` and `response` (or `answer`) per line.
